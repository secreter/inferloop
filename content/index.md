# InferLoop

**Loop to the top.**

工程师写给工程师的 AI Infra 深度内容。每一本书都让你读完后能独立上手干活。

---

## 书架

### [LLM Infra 工程实战](/llm-infra)

面向应用层工程师的 LLM 基础设施系统化学习路径。从 Transformer 到分布式训练，从推理引擎到生产部署。

### [Hermes Agent 源码解读](/hermes-agent)

解剖 Hermes Agent 架构设计，理解 Agent 运行时、Skill 系统、消息路由的核心实现。

### [OpenClaw 源码解析](/openclaw)

现代 Agent 系统的架构设计与工程实践，从 Monorepo 到 Gateway，从 LLM 调度到工具执行。

### [Claude Code Skill 指南](/claude-skill)

掌握 Claude Code Skill 开发，构建可复用的 AI 工程能力模块。

### [Claude 插件官方指南](/claude-plugins)

从开发工具到专业领域，系统掌握 Claude 插件生态。
